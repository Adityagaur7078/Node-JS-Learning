# 🚀 Backend Learning Projects

A collection of backend projects built while learning **Node.js**, **Express.js**, **MongoDB**, **Mongoose**, **JWT Authentication**, **bcrypt**, **Multer**, and other backend technologies.

This repository documents my journey of learning backend development through hands-on projects, starting from the fundamentals and gradually building real-world applications.

## 📖 About

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Adityagaur7078/Backend_Learning_Projects.git
```

Move into the repository

```bash
cd Backend_Learning_Projects
```

Install dependencies

```bash
npm install
```

Run any project

```bash
npm start
```

> Each project has its own dependencies and configuration. Navigate into the desired project folder before installing packages and running it.
